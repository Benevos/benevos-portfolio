## How to run

After cloning the repository, install the necessary dependencies by running:

```bash
npm install
```

And then you can start development server by:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open http://localhost:3000 with your browser to see the result.
